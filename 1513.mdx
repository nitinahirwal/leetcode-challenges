## ğŸ’¡ Intuition

When you see consecutive `1`s:

- A run of length `1` â†’ contributes `1`
- A run of length `2` â†’ contributes `1 + 2 = 3`
- A run of length `3` â†’ contributes `1 + 2 + 3 = 6`

In general:
```
For a run of length L:  
Total substrings = L Ã— (L + 1) / 2
```

This gives a direct counting approach without generating substrings.

## ğŸ“Œ Approach

1. Iterate through the string left to right.
2. Maintain a variable `run` that counts consecutive `1`s.
3. On every `'1'`:
   - Increment `run`
   - Add `run` to the answer
4. On `'0'`, reset `run = 0`
5. Keep everything modulo **1e9+7**.

This converts a potentially O(nÂ²) substring problem into an efficient **O(n)** solution.

```javascript []
/**
 * @param {string} s
 * @return {number}
 */
var numSub = function(s) {
    const MOD = 1000000007;
    let ans = 0;
    let run = 0;
    for (let i = 0; i < s.length; ++i) {
        if (s[i] === '1') {
            run += 1;
            ans = (ans + run) % MOD;
        } else {
            run = 0;
        }
    }
    return ans;
};
```
